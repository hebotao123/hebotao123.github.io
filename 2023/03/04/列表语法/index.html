列表语法

Created2023-03-04|Updated2023-03-04|笔记语言
|Post View:

列表语法可以将多个条目组织成有序或者无序列表。

要创建有序列表，需要在每个列表项前添加数字并紧跟一个英文句点还有空格。数字不必按数学顺序排列，但是列表必须已数字1起始。
例如

  1. 列表1
  2. 列表2
  3. 列表3
  4. 列表4
  5. 列表5

要创建无序列表，需要在每个列表项前加-、*或者+。缩进一个或者多个列表项可以创建嵌套列表。
例如

  • 列表1
    • 列表2
    • 列表3
      • 列表4
    • 列表5

还可以在列表中嵌套其他元素。不过若是要保留列表的连续性的同时在列表中添加另一种元素，需要将该元素缩进4个空格或一个制表符。
例如段落

  • 这是一个段落

  • 这是第二条段落

        这是第三条段落

  • 这还一条段落

代码块

代码块通常使用四个空格或者一个制表符缩进。当他们被放到列表中时，需要使用八个空格或者两个制表符缩进。
例如

  1. Open the file.

  2. Find the following code block on line 21:

    <html>
  <head>
    <title>Test</title>
  </head>

  3. Update the title to match the name of your website.
