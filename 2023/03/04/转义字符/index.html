转义字符

Created2023-03-04|Updated2023-03-04|笔记语言
|Post View:

要显示原本用于格式化markdown文档的字符，需要在字符的前面添加反斜杠字符 \。
例如

\*这是星号

可用做转义的字符:

  • \
  • `
  • *
  • _
  • { }
  • [ ]
  • ( )
  • #
  • +
  • -
  • .
  • !
  • |

特殊字符自动转义

不是很理解
在html文件中，有两个字符需要特殊处理:<和&。<符号用于起始标签，&符号则用于标记html实体，如果你只是想用这些符号，你必须要使用实体的形式，例如&lt和&amp。

markdown允许你直接使用这些符号，它帮助你自动完成转义。如果你使用&符号作为html实体的一部分，那么他不会被转换，而在其他情况下，它则会被转换成&amp;。所以你如果要在文件中插入一个著作权的符号，可以这样写:
&copy;
markdown将不会对这段文字做修改，但如果这样写:
AT&T　　
markdown就会将它转换为:
AT&amp;T
AT&T

类型的情况也会发生在<符号上，因为markdown支持行内html，如果你使用<符号作为html标签的分隔符，那markdown也不会对他进行任何转换，但是如果你是这样写:
5 < 4
markdown将会把它转换为:
4&lt;5

需要特别注意的是，在markdown的块级别元素和內联元素中，<和&两个符号都会被自动转换成html实体，这个项特性可以很容易的使用markdown写html。(在html语法中，你需要手动把所有的< &都转换成html实体。)
